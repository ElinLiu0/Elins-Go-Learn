运算符介绍：
    --运算符是一种特殊的符号，用于表示数据的运算、赋值和比较等
    1）算术运算符
        介绍：对数值型的变量进行运算的，比如加减乘除
        除法案例：
            package main

            import "fmt"

            func main() {
                //重点讲解 /和%
                /*说明：如果运算的数都是整数，
                那么相除的结果自动去掉小数部分，
                并保留整数部分*/
                fmt.Println(10 / 4)
                /*之所以会输出值为2是因为10和4均为int
                而Go编译器为了保证数据结构一致因此输出值
                的类型也应当为int，所以输出值为2
                */
                /*将值添加.0后输出正常*/
                fmt.Println(10.0 / 4.0)
            }
        细节说明：
            1）对于除号"/"，整数之间支部会保留整数部分而不会保留小数
            2）当对一个数取余时，可以等价于a%b = a-a/b*b
            3）GoLang的自增自减是无法赋值的，比如 a = i++是不被允许的
            4）GoLang的++和--只能写在变量后面，不能写在变量的前面，
            即：只有a++和a--但是没有++a和--a
            5）GoLang的设计者去掉C/Java中的自增自减的容易混淆的写法，让
            GoLang更加简洁，统一。
